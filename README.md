# Jaime P. Lepe Math Tutoring

A one-page tutoring website for Jaime P. Lepe.

## Publish with GitHub Pages

1. Create a GitHub account at github.com if needed.
2. Create a new **public** repository named `jaime-tutoring`.
3. Upload all files from this folder to the repository.
4. In the repository, open **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/ (root)` folder, then click **Save**.
7. Your site will appear at:

   `https://YOUR-USERNAME.github.io/jaime-tutoring/`

Replace `YOUR-USERNAME` with your GitHub username.
